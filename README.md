# ProjetNarratif

## Description: 
-Jeux narratif où vos choix détermineront votre propre histoire. Écrivez les mots présents dans les [ ] pour effectuer une action.

## Fonctionnalités implémentées: 
-Utilisation de "Room" sur Visual Studio
-Différents résultats implémentées selon les choix du joueur
-Dialogue/Histoire en temps réel

## Tech Stack: 
-Visual Studio

## Installer / Jouer: 
- Se rendre sur mon portfolio itch (https://abdou-24.itch.io/)
- Cliquez sur le projet et installer le dossier.zip
- Décompresser le dossier et l'ouvrir
- Lancer le fichier ProjetNarratif.exe
